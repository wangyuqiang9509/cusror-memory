# 构建与修复

逐步修复 TypeScript 及构建错误：

1. 执行构建：`npm run build` 或 `pnpm build`

2. 解析错误输出：
   - 按文件分组
   - 按严重程度排序

3. 针对每个错误：
   - 显示错误上下文（前后各 5 行）
   - 解释问题原因
   - 提出修复方案
   - 应用修复
   - 重新运行构建
   - 验证错误已解决

4. 以下情况立即停止：
   - 修复引入了新错误
   - 同一错误连续尝试 3 次仍未解决
   - 用户请求暂停

5. 输出汇总报告：
   - 已修复的错误数
   - 剩余的错误数
   - 新引入的错误数

**每次只修复一个错误，确保安全！**
