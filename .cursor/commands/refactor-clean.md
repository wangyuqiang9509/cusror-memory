# 重构清理

安全识别并移除死代码，全程测试验证：

1. 执行死代码分析工具：
   - knip：检测未使用的导出与文件
   - depcheck：检测未使用的依赖
   - ts-prune：检测未使用的 TypeScript 导出

2. 在 `.reports/dead-code-analysis.md` 生成完整分析报告

3. 按风险等级分类：
   - **安全**：测试文件、未使用的工具函数
   - **谨慎**：API 路由、组件
   - **危险**：配置文件、主入口文件

4. 仅提议安全的删除操作

5. 每次删除前执行：
   - 运行完整测试套件
   - 确认测试通过
   - 应用变更
   - 重新运行测试
   - 测试失败则立即回滚

6. 展示清理结果汇总

**切勿在未运行测试的情况下删除任何代码！**
