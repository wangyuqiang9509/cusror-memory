# start-stop

**该指令用于给当前项目创建一键启动和暂停服务脚本**

## 一键启动与暂停服务脚本使用说明

### 启动服务脚本

```bash
./start.sh
```

- 使用智能构建（检测代码变更）
- 保留依赖缓存，提高构建速度
- 适用于日常开发

### 强制重新构建

```bash
./start.sh --force
```

- 清除所有构建缓存
- 完全重新下载和构建
- 适用于依赖更新或重大变更

## 暂停服务脚本

### 基本停止（保留数据和镜像）：

```bash
./stop.sh
```

### 停止并清除数据库数据：

```bash
./stop.sh -v
```

### 停止并删除镜像（下次启动需要重新构建）：

```bash
./stop.sh -i
```

### 完全清理（停止 + 删除数据 + 删除镜像）：

```bash
./stop.sh -a
```
